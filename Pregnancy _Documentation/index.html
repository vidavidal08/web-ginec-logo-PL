Installation

Follow the steps below to get started with your Site Template:

  1. Open the Pregnancy_Main_File Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • Pregnancy_HTML/css - Extra Stylesheets Folder
    • Pregnancy_HTML/fonts - Fonts Family
    • Pregnancy_HTML/images - Images Folder
    • Pregnancy_HTML/js - Javacripts Folder
    • Pregnancy_HTML/vendor - Plugin Files
    • Pregnancy_HTML/index.html - Index File/Homepage
    • Pregnancy_HTML/book-appointment.php - Appointment Form
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

HTML File

The html template uses Bootstrap 3 with valid HTML5 tags. This Template is a responisve layout with one column. All of the information in content area is nested within a class and comes with predefined classes.

  • index.html - Index File/Homepage
